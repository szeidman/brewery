# README

This is an application for managing beer recipes and ingredients. Users without accounts can view beers and ingredients. Users with accounts can create new ingredients and beers, and edit and delete beers they have created.

The application uses a Rails API with ActiveModel serializers. Portions of the app use a jQuery front end.

* Ruby version: ruby 2.3.3p222

* To use: Run bundle install and rake db:migrate. For Facebook OAuth integration you will need a key and secret from Facebook's developers site.
